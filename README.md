# 🛍️ Fashion Store Website

A modern and responsive **Fashion Store Website** built using HTML, CSS, and JavaScript.
This project represents an online shopping platform with stylish UI, product categories, and user interaction features.

---

## 🚀 Features

* 🏠 Attractive Home Page with banner section
* 🛒 Product listing for Men & Women
* 🧥 Categories section with offers & deals
* 🔐 Login system UI
* 📱 Fully responsive design
* 🎨 Clean and modern user interface

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
